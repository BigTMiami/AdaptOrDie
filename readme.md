# Organization
The following folders hold the code base for paper
* dataset_creation
    * Contains the scripts and notebooks used to create the datasets used in the experiments
* experiments_in_paper
    * Contains all of the notebooks used to create results in the paper
* latex
    * Contains the latex files plus png figures of the paper
* dont-stop-pretraining-modified
    * The code base from Gururangan et al. 2020 that needed to be adjusted to run on current software
* exploratory_experiments
    * Experiments that were used in developing insight, but not directly included in the paper
